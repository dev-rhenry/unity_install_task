## What I Did

## Unity Interface Overview

* Scene View: Used to build the game world
* Hierarchy: Lists all objects in the scene
* Inspector: Shows properties of selected objects
* Project Window: Contains assets and files
* Console: Displays logs and errors

## My First Impression

Unity interface is powerful but slightly complex at first. After exploring, I understood how different panels work together.
